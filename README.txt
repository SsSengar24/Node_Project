REST API

GET /users - get data for all users

GET /users/:id - gets the data for a given user (:id --> dynamic variable)
Eg: GET /users/1 - gets data for user 1
    GET /users/2 - gets data for user 2


POST /users - create a new user

PUT /users/:id - edits the given user
EG: PUT /users/1 - edits the data for user 1
    PUT /users/2 - edits the data for user 2

DELETE /users/:id - deletes a given user
Eg: DELETE /users/1 - deletes user 1
    DELETE /users/2 - deletes user 2
